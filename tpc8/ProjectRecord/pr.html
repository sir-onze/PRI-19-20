Abstract:

Para este trabalho de casa foi proposta a criação uma aplicação em NodeJS capaz de receber pedidos GET e de responder a página pretendida pelo cliente e a POST de maneira a permitir a inserção de ficheiros na base de dados. Sendo que a web app se alimenta de uma API de dados de maneira a satisfazer a arquitetura estudada nas aulas.

O servidor criado corre na porta 3012 e responde a pedidos com a seguinte formatação de exemplo: localhost:5018.
