Abstract:

Para este trabalho de casa foi proposta a criação um servidor em NodeJS capaz de receber pedidos GET e de responder com a página html inical e também individual de um filme.Como base de dados foi utilizado uma base de dados em mongo disponibilizada pelo docente. Para além disso teria de haver capacidade para responder a pedidos DELETE, para apagar um filme específico , assim como a pedidos POST para inserção de filmes.

De notar que a aplicação está dividida em duas partes, a aplicação web responsável por apresentar as informações com interface visual aos utilizadores e a API, que apenas responde aos pedidos na forma de dados. A aplicação web foi implementada de maneira a utilizar a API para obter os dados necessários para o seu funcionamento.

O servidor criado corre na porta 3006 e responde a pedidos com a seguinte formatação de exemplo: localhost:3005, sendo que neste caso disponibilizaria a página inicial, definida como index,com as opções de aceder à listagem dos filmes ou de inserir um filme .
